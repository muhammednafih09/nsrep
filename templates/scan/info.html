VERSION INFO

        NSREP is a Network Scanning and Reporting System or Tool, that helps the network penetration testing and create reports according to the 
        details found in the scanning. This project is based on the scanning phase in the ethical hacking phases.NSREP network scanning and reporting 
        system is working along with a well-known network scanning tool called Nmap. This allows the team lead to monitor the activities history 
        of the other team members.
